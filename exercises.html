Exercises

This page holds some suggested worksheets for using this extension. These worksheets should be suitable for people that have never used Scratch before, although so some extent, some familiarity may be assumed at certain points. The exercises are designed to challenge, and to inspire different ways about solving problems. They are in the style of code club worksheets, and as such contain all of the code on the respective files, these work well for a computer-club based activity, but of course if being used for teaching, the worksheets should instead be used as a guideline for the teacher.
